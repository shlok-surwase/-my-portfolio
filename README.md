# Shlok Surwase — Portfolio Website

A premium dark-themed personal portfolio website built with HTML, CSS, and JavaScript.

## 🌐 Live Site
[View Portfolio](https://your-username.github.io/portfolio)

## ✨ Features
- Dark / Light theme toggle
- Custom animated cursor
- Scroll-reveal animations
- Typing effect hero section
- Parallax scrolling
- Interactive gallery with lightbox
- Fully responsive (mobile, tablet, desktop)
- Contact form with validation

## 📁 Sections
- **Home** — Hero with photo, name, introduction
- **About Me** — Education, goals, strengths, skills
- **Projects** — RoseAI, Velocity Arena
- **Achievements** — Cisco, Coursera, Disha certificates
- **Gallery** — All photos and certificates
- **Contact** — Email, GitHub, LinkedIn, phone

## 🛠 Tech Stack
- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (IntersectionObserver, custom cursor, theme toggle)

## 🚀 Deploy on GitHub Pages
1. Upload all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at `https://your-username.github.io/repo-name`
